# Extension Icons

This folder should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (Windows)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any image editor. The icons should:

1. **Be square** (same width and height)
2. **Have a transparent background**
3. **Be simple and recognizable** at small sizes
4. **Use the touchpad/mobile theme** (üì± or üñ±Ô∏è symbols work well)

## Suggested Design

A simple design could be:
- Background: Blue circle (#007bff)
- Icon: White phone or touchpad symbol
- Style: Modern, flat design
